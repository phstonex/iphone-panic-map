V18 维修标准（内部通用）

① 电池标准

  • 健康度 低于 85% → 一律更换；更换后目标 ≥ 95%
  • 实际容量或内阻衰退明显，健康度 低于 70% → 视为电芯报废，建议更换电芯级别处理

② 后摄检测流程

  • 优先确认是否可清洁修复（镜组、 OIS、模组座）
  • 区分：单摄损坏 / 整组损坏（三摄）
  • 能清洁就清洁，能修复就修复；确属损坏再更换模组

③ 屏幕更换机型要求

  • iPhone 13 Pro / 13 Pro Max → 必须使用 OLED 屏
  • 翻新机型 → 可更换 LCD + IC 方案

④ 功能完整性标准

  • 如遇难以修复的功能：Face ID / RT / 传感等 → 必须备注记录
  • 未备注的设备默认 全功能正常
  • 重点：最终交付必须“全功能修好”
